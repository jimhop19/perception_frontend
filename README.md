# Perception
> Perception 是一個台灣新聞的比較平台。
> <br>
> 藉由不同立場的媒體報導，更客觀地理解新聞事件。
> <br>
> 透過媒體發佈新聞的時間軸，更全面地掌握事件脈絡。
> <br>
>> 展示網址：https://perception-sigma.vercel.app/
